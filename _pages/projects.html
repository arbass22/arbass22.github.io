Below are a few of the projects that I have been working on in the free time that I have. Each is marked with some of the technologies that were used as well as links to the repositories and demos.
PoliTweet
At HackBeanpot 2016, I worked to build a web app that analyzed tweets about the 2016 presidential primary candidates. We used Indico's sentiment analysis to determine how positive or negetive each tweet was. We then used D3 to graph the changes in sentiment over time.
APIs: Twitter Streaming, Indico
Techonologies: NodeJS, ReactJS, MongoDB

GaragePi
I built an internet-connected garage door opener for my home. The project used a Raspberry Pi that controlled the doors/sensors and a NodeJS server to handle the authentication of the user and the dispatching of requests to the Pi.
Techonologies: Raspberry Pi, NodeJS, Web Sockets
{% include github-button.html url="https://github.com/arbass22/internet-garage"%}
Embedded Security Capture the Flag
I participated in the 2016 Collegiate eCapture-the-Flag competition hosted by MITRE centering around embedded security. Our task was to build a secure implementation of an internet-connected door look with two factor authentication. The opening device used a BeagleBone Black with a number keypad. We had six weeks to build our system and six weeks to try to break into a system provided by a competing school's team.
Techonologies: Twisted, BeagleBone Black
